    Penjelasan Tipe Fungsi
    Bahu Kiri (Left Shoulder)

    Nilai maksimum di kiri (≤a), menurun linear hingga b, lalu 0. Contoh: "baru", "jarang", "sempit"

    Formula: μ(x) = (b-x)/(b-a)

    Segitiga (Triangle)

    Nilai 0 di a, naik ke puncak di b, turun ke 0 di c. Contoh: "sedang", "cukup"

    Formula: μ(x) = (x-a)/(b-a) atau (c-x)/(c-b)

    Bahu Kanan (Right Shoulder)

    Nilai 0 di kiri (≤a), naik linear hingga b, lalu maksimum. Contoh: "lama", "sering", "luas"

    Formula: μ(x) = (x-a)/(b-a)
